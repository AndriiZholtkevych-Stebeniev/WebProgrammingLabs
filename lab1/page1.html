Розробка простої веб-сторінки з використанням HTML5

World Wide Web (WWW, web, веб, Всесвітня павутина) -це розподілена інформаційна

система, яка надає доступ до гіпертекстових документів через протокол HTTP (HyperText

Transfer Protocol). WWW - це мережева технологія, яка побудована на клієнт-серверній архітектурі та

використовує інфраструктуру Інтернет для взаємодії між сервером та клієнтом (мал. 1).
