Завдання: Розробити веб-сторінку структура

якої визначається використанням

тегів мови розмітки: section, header, footer,

article.

Для оформлення мультимедійних елементів

на сторінці можна використовувати теги: figure, figcaption,

video, audio.

Для оформлення мультимедійних елементів на сторінці використовувати теги:

pre, code, p strong, b, a.
